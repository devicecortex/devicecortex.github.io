zCastle

Smart Home Control for Android

Available on Android

Smart Home Control

Conveniently control all of your smart home devices via the Homeseer, Vera, or Wink Hub 2 home automation gateway.

Supported Devices
  • Homeseer v3
  • Vera gateways running UI6/UI7
  • Wink Hubs running API v2
Features
  • Run scenes (Wink shortcuts)
  • Control thermostats
  • Control lights (dimmers and on/off switches)
  • View sensor states (temperature, motion, open/close)
  • Control door locks
  • Create Android widgets for quick access
  • Create custom dashboards
Homeseer Capabilities
  • Remote access via MyHS service
Vera Capabilities
  • Local and remote network access
  • Change Vera modes
  • PIN code support for mode changes
  • Arm/disarm sensors
  • View cameras and control Pan/Tilt
  • Save camera images
Wink Capabilities
  • Wink Hub device control via Wink cloud access
