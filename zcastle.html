Smart Home Control

Conveniently control all of your smart home devices via the Homeseer, Vera, or Wink Hub 2 home automation gateway.

zCastle is available on both the Amazon App Store and Google Play Store

Supported Devices

  • Homeseer v3
  • Vera gateways running UI6/UI7
  • Wink Hubs running API v2

Features

  • Run scenes (Wink shortcuts)
  • Control Thermostats
  • Control Lights (Dimmers and Simple on/off switches)
  • View sensors current state (temperature, motion, open/close)
  • Control door locks
  • Create Android widgets for quick access to modes and scenes
  • Create custom dashboards to organize devices

Homeseer Specific Capabilities

  • Remote access via MyHS service

Vera Specific Capabilities

  • Local and Remote network access to control smart devices via Vera gateway
  • Change Vera modes
  • Support for PIN codes to unlock app when change Vera mode back to Home
  • Arm/Disarm Sensors
  • View Cameras and control Pan/Tilt
  • Save camera images

Wink Specific Capabilities

  • Wink Hub device control via Wink cloud access
